# SIGEF - Sistema de Gestão de Funcionários

## Visão Geral
Este projeto é um Sistema de Gestão de Funcionários (SIGEF) desenvolvido como aplicação desktop. O sistema permite aos administradores gerir informações de funcionários, incluindo dados pessoais, cargos e gestão de salários. Possui sistema de login, dashboard com estatísticas e operações CRUD completas para dados de funcionários.

## Tecnologias Utilizadas
- **Java** - Linguagem de programação principal
- **JavaFX** - Framework de interface gráfica
- **FXML** - Marcação de interface baseada em XML
- **MySQL** - Gestão de base de dados
- **SQL** - Consultas à base de dados

## Aplicações Utilizadas
- **Scene Builder** - Para criar o layout da aplicação
- **Apache NetBeans IDE 24** - Para desenvolvimento do código

## Funcionalidades
- Sistema de autenticação de utilizadores
- Dashboard com estatísticas de funcionários
- Registo, atualização e eliminação de funcionários
- Gestão de salários
- Funcionalidade de pesquisa de funcionários
- Upload de imagens para perfis de funcionários
